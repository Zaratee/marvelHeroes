# MarvelHeroes
_Created with CodeSandbox_ <br><br>
Aplicación Marvel Héroes, donde se muestra una lista de héroes acomodados por orden alfabeto y al clickear un héroe lo agregaras como favorito.
Haciendo uso de:
- Redux
- css
- ApiMarvel (con ApiKey)
- Fetch
